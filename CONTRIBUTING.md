# Contributing to LinkedIn Stalker

Thank you for your interest in contributing to LinkedIn Stalker! We welcome contributions from the community.

## How to Contribute

### Reporting Bugs

If you find a bug, please open an issue with:
- A clear description of the problem
- Steps to reproduce the issue
- Expected vs actual behavior
- Your environment (OS, Python version, Playwright version)

### Suggesting Features

Feature suggestions are welcome! Please:
- Check if the feature has already been requested
- Clearly describe the feature and its use case
- Explain how it would benefit users

### Pull Requests

1. **Fork the repository** and create a new branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```

2. **Make your changes**:
   - Follow the existing code style
   - Add comments for complex logic
   - Update documentation if needed

3. **Test your changes**:
   - Ensure the scraper still works
   - Test with multiple companies/URLs
   - Verify output format is correct

4. **Commit your changes**:
   ```bash
   git commit -m "Add: description of your feature"
   ```

5. **Push to your fork**:
   ```bash
   git push origin feature/your-feature-name
   ```

6. **Open a Pull Request**:
   - Provide a clear description of changes
   - Reference any related issues
   - Include screenshots if UI/output changes

## Code Style

- Follow PEP 8 for Python code
- Use descriptive variable names
- Add docstrings to functions
- Keep functions focused and modular

## Areas for Contribution

- **Selectors**: Improve LinkedIn HTML selectors for better reliability
- **Error Handling**: Add better error messages and recovery
- **Features**: Add new scraping capabilities (comments, reactions, etc.)
- **Performance**: Optimize scraping speed and resource usage
- **Documentation**: Improve guides and examples
- **Testing**: Add unit tests and integration tests

## Questions?

Feel free to open an issue for any questions about contributing!

## Code of Conduct

Be respectful and constructive. We're all here to build something useful together.
