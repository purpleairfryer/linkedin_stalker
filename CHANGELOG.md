# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-05

### Added
- Initial release
- LinkedIn post scraper using Playwright
- Cookie-based authentication system
- Multi-company support with organized output
- Duplicate detection and repost filtering
- URN-based post identification
- Debug mode for troubleshooting
- Comprehensive documentation

### Features
- Extract top N posts from company pages and user feeds
- Organize output by company with clear visual separators
- Support for multiple URLs in batch processing
- Flexible selector strategies for different LinkedIn layouts
- Clean text extraction with metadata filtering

### Documentation
- README.md with setup and usage instructions
- LINKEDIN_STRUCTURE.md with LinkedIn HTML documentation
- CONTRIBUTING.md with contribution guidelines
- Example configuration files
- MIT License

## Future Enhancements

Potential features for future versions:
- Support for comments and reactions
- Export to JSON/CSV formats

- Better date parsing for multiple languages
